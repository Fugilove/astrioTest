# Vue Test - Каталог товаров

Тестовое задание для Astrio на Vue 3 с использованием Vuetify и Pinia.

## Технологии

- Vue 3 (Composition API, script setup)
- Vuetify 3
- Pinia (управление состоянием)
- Vue Router
- Vite

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Реализованные функции

### Level 1 (Обязательные)
- ✅ Список товаров с фильтрацией по брендам
- ✅ Добавление товаров в корзину
- ✅ Счетчик товаров в корзине
- ✅ Кастомный дизайн с использованием Vuetify

### Level 2 (Необязательные)
- ✅ Респонсивный дизайн (от 320px)
- ✅ Страница корзины с возможностью изменения количества
- ✅ Удаление товаров из корзины
- ✅ Пересчет суммы заказа в реальном времени
- ✅ Навигация между страницами (Vue Router)

### Level 3 (Необязательные)
- ✅ Настраиваемые товары (configurable) с опциями (цвет, размер)
- ✅ Валидация выбора опций перед добавлением в корзину
- ✅ Изменение изображения товара при выборе варианта
- ✅ Отображение выбранных опций в корзине

## Структура проекта

```
src/
├── assets/
│   ├── data/          # JSON файлы с данными
│   └── styles/        # Кастомные CSS стили
├── components/        # Vue компоненты
├── plugins/          # Плагины (Vuetify)
├── router/           # Конфигурация роутера
├── stores/           # Pinia stores
└── views/            # Страницы приложения
```

## Особенности реализации

- Использование `<script setup>` синтаксиса во всех компонентах
- Pinia для управления состоянием корзины и продуктов
- Vuetify компоненты для UI
- Кастомные CSS стили для уникального дизайна
- Поддержка настраиваемых товаров с вариантами
